QiitaSearch
====

Qiita APIを利用して、投げたクエリに対する検索結果の一覧をcsvに出力します。

## Usage

*main.py* を編集し、以下を設定してください。

* **API_KEY** : 取得したQiita APIキー
* **SEARCH_QUERY** : 検索したい文字列

プログラムを実行すると、検索結果のリストが *"Output/日時_検索文字列.csv"* に出力されます
